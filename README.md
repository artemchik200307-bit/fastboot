# FASTBOOT Static Website

Первая самостоятельная версия торгового сайта без Telegram и без backend.

## Возможности

- адаптивный интерфейс;
- свечной demo-график;
- BTC, ETH, EURUSD, GBPUSD, XAUUSD и NAS100;
- таймфреймы 5m, 15m, 1H, 4H и 1D;
- demo-анализ LONG/SHORT;
- Entry, Stop Loss, Take Profit и RR;
- зоны спроса и предложения;
- риск-калькулятор;
- торговый журнал через localStorage;
- полностью статический сайт;
- подходит для бесплатного GitHub Pages.

## Локальный запуск

Можно просто открыть `index.html`.

Лучше открыть через локальный сервер:

```bash
python -m http.server 8000
```

Затем:

```text
http://localhost:8000
```

## Размещение на GitHub Pages

1. Создайте новый GitHub-репозиторий.
2. Загрузите в корень:
   - `index.html`
   - `styles.css`
   - `app.js`
3. Откройте `Settings`.
4. Перейдите в `Pages`.
5. В `Build and deployment` выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Нажмите `Save`.
7. Через несколько минут GitHub покажет адрес сайта.

## Ограничения этой версии

- котировки демонстрационные;
- анализ работает локально;
- журнал хранится только в текущем браузере;
- нет аккаунтов;
- нет backend;
- нет реальных ордеров;
- нет Telegram-авторизации.

Следующим этапом можно подключить реальные данные и backend, не меняя основную структуру интерфейса.
