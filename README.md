# FASTBOOT — главная страница рынка

## Изменения

- График полностью удалён с главной страницы.
- Главная страница показывает топ‑100 активных криптовалютных USDT-пар.
- Добавлен отдельный блок «Топ роста».
- Добавлен отдельный блок «Топ падения».
- При выборе инструмента обновляется карточка статистики рынка.
- Статистика включает цену, изменение, максимум, минимум, объёмы,
  средневзвешенную цену, цену открытия и количество сделок.
- Бегущая строка с котировками сохранена.
- Кнопки входа и регистрации сохранены.
- Рыночные данные обновляются автоматически каждую минуту.

## Обновление сайта

В приватном GitHub-репозитории `fastboot` замените:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`

После Commit Render автоматически развернёт новую версию.
