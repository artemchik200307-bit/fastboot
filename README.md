# FASTBOOT — реальные криптовалютные графики

Отдельный статический сайт FASTBOOT. Никакого Telegram-бота или старого проекта
в этой версии нет.

## Что изменено

- Оставлены только криптовалюты.
- Полностью удалены Forex, индексы, акции и товары.
- Полностью удалён анализ рынка и торговые рекомендации.
- Исторические свечи загружаются из публичного Binance Spot API.
- Текущая свеча обновляется через Binance WebSocket.
- Цены и изменение за 24 часа реальные.
- API-ключ Binance не требуется.
- Сохранён риск-калькулятор.
- Сохранён локальный журнал.

## Файлы для GitHub

Замените в приватном репозитории `fastboot` три файла:

- `index.html`
- `styles.css`
- `app.js`

После Commit Render автоматически выполнит новый deploy.

## Источник данных

REST:

```text
https://data-api.binance.vision/api/v3
```

WebSocket:

```text
wss://stream.binance.com:9443/ws
```

Используются только публичные рыночные данные. Торговля и операции с аккаунтом
не подключены.

## Важно

Некоторые интернет-провайдеры или страны могут ограничивать доступ к Binance.
В этом случае сайт покажет ошибку подключения. Позже можно добавить резервный
источник данных.
