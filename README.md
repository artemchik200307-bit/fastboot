# FASTBOOT — главная страница

Новая главная страница отдельного проекта FASTBOOT.

## Изменения

- FASTBOOT расположен по центру верхней панели.
- Справа добавлены кнопки входа и регистрации.
- Удалены информационные блоки «Источник данных», «Режим» и «Торговля».
- Удалены рекламный текст и кнопка «Открыть рынок».
- График остался интерактивным.
- Сохранены выбранный инструмент, цена, таймфрейм и список инструментов.
- Сохранена статистика рынка.
- Удалены риск-калькулятор и торговый журнал.
- Список формируется динамически из топ‑100 активных криптовалютных USDT-пар
  по суточному объёму торгов.
- Добавлена бегущая строка с ценами и изменением за 24 часа.
- Котировки обновляются автоматически.

## Обновление сайта

В приватном GitHub-репозитории `fastboot` замените:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`

После Commit Render автоматически развернёт новую версию.
